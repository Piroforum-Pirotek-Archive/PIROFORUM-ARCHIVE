А вот насчёт вытяжки не соглашусь. Потому что есть возможность из NO2 восстановить HNO3.   
Сначала так:
                            3NO2 +  H2O = 2HNO3 + NO
После чего:     
                            2NO + O2 = NO2 н.у.
Круг замкнулся.
В сумарном виде это выглядит так:

                            2H2O + 4NO2 + O2 = 4HNO3
Таким образом для производства безконечного количества (при 100% выходе) серной кислоты из безконенечного количества серы нужно немного кислоты. Т.К. Её можно восстанавливать указаным способом. Именно таким образом в кислотные дожди попадает азотка.
