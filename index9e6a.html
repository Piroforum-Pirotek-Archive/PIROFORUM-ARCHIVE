Для нейтрализации 1 моль NaOH требуется 1 моль HCl
Найдем количество вещества NaOH
М( NaOH) = 23 + 16 + 1 =40 г/моль
n ( NaOH) = 20 : 40 = 0,5 моль - NaOH вступило в реакцию
n ( NaOH) = n (HCl) = 0,5 моль
М (HCl) = 1 + 35,5 = 36,5 г/моль
m (HСl) = 3635 * 0,5 = 18,25 г - соляной кислоты потребуется для нейтрализации
18,25 г - 20%
х г - 100%
х = 18,25*100:20 = 91,25 г - 20%-ной HCl потребуется
А вот чтобы найти количество миллилитров HCl, надо знать ее плотность, найдите в справочнике плотность и умножьте на последнюю массу - найдете искомый ответ
