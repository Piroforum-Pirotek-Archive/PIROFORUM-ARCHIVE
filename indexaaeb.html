вышибного берется в зависимости от 3 факторов
 1) скорость горения пороха (зависит сколько стараний приложено для его изготовления)
 2) массы снаряда
 3) калибра
 для хорошего пороха соотношение массы снаряда до вышибного 1/10-1/12
 для крупных калибров можно брать 1/15 (6 "и выше)
 так что подбирать вышибной надо каждый раз после новой партии пороха
 выше приведенные соотношения для пороха хорошего качества для которого не пожалели ни времени ни сил на его изготовление
