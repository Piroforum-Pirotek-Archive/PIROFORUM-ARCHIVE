Посчитай объём забитого корпуса, учитывая объём трубочки для детона, зная точный вес, посчитай какая плотность выходит, для ориентира на будущее. Трубчатые шашки удобны для закладки в просверленные по диаметру отверстия(круглые), для подрыва брёвен, бетона и т.д., накладным зарядом, эффективней с плоской поверхностью, для более плотного контакта с разрушаемым объектом, больше КПД. Лучше делай прямоугольные(как кусок мыла ) или квадратные, их можно плотно наращивать друг к другу столько, сколько необходимо.
